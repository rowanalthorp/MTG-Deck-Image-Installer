# Welcome to: **magic-the-tabletop-simulator-deck-generation** (long name I know...) 
<br>
This applications takes your exported deck lists from a site like [Archidekt](https://archidekt.com/) and generates a working JSON that is ready to be imported into the Table Top Simulators Saved Objects Folder (~Here: \Documents\My Games\Tabletop Simulator\Saves\Saved Objects) ready for use.
